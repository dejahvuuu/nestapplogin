## Login - NestJS

Una API RESTful básica para realizar un login con JWT, desarrollada con [NestJS](https://nestjs.com/). 

## Clonar el proyecto

```bash
git clone https://github.com/dejahvuuu/nestapplogin.git
```

## Intalación de dependencias y compilación 

```bash
$ npm install
```

```bash
$ npm run start
``
